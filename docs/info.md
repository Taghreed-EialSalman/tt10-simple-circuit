# Simple Circuit: F = AB + C'

## Project Overview
This project implements a simple combinational logic circuit using TinyTapeout.

## Functionality
The circuit computes:
F = (A AND B) OR (NOT C)

## Inputs and Outputs
- Inputs:
  - A (ui[0])
  - B (ui[1])
  - C (ui[2])
- Output:
  - F (uo[0])

## Notes
- Purely combinational design
- No clock required
